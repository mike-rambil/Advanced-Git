# git maintenance start


#### Category
Maintenance
> Runs a cronJob in background for the specified repo for periodic maintenance.


#### Command
```sh
git maintenance start
```

#### Examples
- **Enable background maintenance for your repository.**


```sh
git maintenance start
```


#### Steps
1. Run `git maintenance start` in your repository.
