# Check Existing Worktrees


![Category: Worktree](https://img.shields.io/badge/Category-Worktree-blue)

#### Command
```sh
git worktree list
```

#### Examples
- **List all active worktrees.**


```sh
git worktree list
```
- **List worktrees in a machine-readable format.**


```sh
git worktree list --porcelain
```
