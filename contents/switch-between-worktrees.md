# Switch Between Worktrees


![Category: Worktree](https://img.shields.io/badge/Category-Worktree-blue)

#### Command
`cd <worktree-path>`

#### Examples
- **Switch to the worktree directory.** 

 ```sh
cd ../feature-branch 
 ```
- **Switch to a hotfix worktree.** 

 ```sh
cd ../hotfix 
 ```


#### Steps
1. Simply cd into the worktree directory to switch.
