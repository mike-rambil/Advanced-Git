# Show Commit History of a Specific File


#### Category
History

#### Command
```sh
git log --oneline -- filename.txt
```

#### Examples
- **List all commits that modified `filename.txt`.**

  ```sh
git log --oneline -- filename.txt
```


#### Steps
1. Lists all commits that modified `filename.txt`.
